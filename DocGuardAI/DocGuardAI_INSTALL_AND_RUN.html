Install and Run DocGuard AI

Version: v1.2.0 RC1 U6.1

Status: Production release approved

Recommended Install

Run the approved installer:


release\DocGuardAISetup.exe

The installer is built from:


dist-docguard-ui-polish\DocGuardAI

Installer behavior:

  • Product name: DocGuard AI
  • Executable: DocGuardAI.exe
  • Default install root: {autopf}\DocGuard AI
  • Privileges: non-admin capable
  • Desktop shortcut: current user when running without elevation

Portable Launch

For portable validation or controlled file-share deployment:


dist-docguard-ui-polish\DocGuardAI\DocGuardAI.exe

Do not use older dist-* folders for production validation.

Source Launch

Use source launch only for engineering validation.


python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -e .
sanitize-gui

Alternative source launch:


py main.py
py -m sanitize

Command-Line Examples


sanitize about
sanitize plugins
sanitize license
sanitize scan example.docx --profile aggressive --summary
sanitize clean example.docx --profile aggressive --html-report auto
sanitize run "C:\Source" --profile aggressive --html-report auto
sanitize history --last 20
