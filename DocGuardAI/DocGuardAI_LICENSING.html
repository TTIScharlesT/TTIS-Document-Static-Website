DocGuard AI Licensing

Version: v1.2.0 RC1 U6.1

DocGuard AI supports local trial operation, offline activation for controlled environments, and optional online activation through TSG License Manager.

Product Identity

| Field | Value |

|---|---|

| Product name | DocGuard AI |

| Product ID | docguard_ai |

| Product key prefix | DGA |

| Edition | Core |

Writing Assistant is included in Core and is not separately licensed.

Product Key Format


DGA-XXXX-XXXX-XXXX

Legacy SSP-... keys may be accepted only for RC compatibility.

Check License Status


sanitize license

In the desktop app, use:


Tools -> License Status

Offline Activation


sanitize license --activate DGA-XXXX-XXXX-XXXX

Offline activation stores a local license cache under:


%APPDATA%\DocGuardAI\license.json

Online Activation


sanitize license --activate DGA-XXXX-XXXX-XXXX --online --server-url http://127.0.0.1:8010

Online activation, refresh, and deactivation use TSG License Manager when configured.

Online license cache:


%APPDATA%\DocGuardAI\tsg_license.json

Refresh


sanitize license --refresh --server-url http://127.0.0.1:8010

Deactivate


sanitize license --deactivate
sanitize license --deactivate --online --server-url http://127.0.0.1:8010

Security Notes

  • Do not include full product keys in screenshots or support tickets.
  • Core document processing does not require cloud upload.
  • Licensing and update-check requests contact TSG License Manager only when configured or explicitly invoked.
