DocGuard AI Quick Start

Version: v1.2.0 RC1 U6.1

Status: Production release approved

Install

Use the approved installer:


release\DocGuardAISetup.exe

The installer supports non-admin installation and installs DocGuard AI for the current user when elevation is not available.

For portable validation, launch:


dist-docguard-ui-polish\DocGuardAI\DocGuardAI.exe

Recommended Desktop Workflow

1. Open DocGuard AI.

2. Select a source file or folder.

3. Choose a profile: Standard, Aggressive, or Paranoid.

4. Keep Backup enabled for normal work.

5. Use Dry Run for previews.

6. Click Scan.

7. Review Dashboard, File Review, Findings, and Risk Map.

8. Click Run Cleanroom when ready to clean, verify, report, and record activity.

9. Review Reports and Activity before sharing cleaned output.

Dashboard Drill-Downs

Dashboard cards open the matching detail page:

| Card | Opens |

|---|---|

| Files Scanned | File Review |

| Open Findings | Findings |

| Highest Risk | Risk Map |

| Cleaned Files | Reports |

CLI Quick Commands


sanitize about
sanitize plugins
sanitize license
sanitize scan "C:\Path\To\Source" --profile aggressive --summary
sanitize run "C:\Path\To\Source" --profile aggressive --html-report auto
sanitize history --last 10

Writing Assistant

Writing Assistant is included in the Core package.


sanitize analyze-text --text "Paste text here." --heatmap
sanitize rewrite --text "Paste text here." --style executive
